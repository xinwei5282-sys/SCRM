销售运营过程监控看板 PRD v2.5 · 2026-04-22

1. 需求概述
背景
业务负责人需要在一页内完成 4 个判断：整体盘面是否健康 / 转化断点在哪 / 哪些销售该冲该救 / 今天该推动哪些管理动作
产品目标
让负责人在 3 分钟内看清当前期次整体经营状态
快速定位链路断点和成交贡献结构
直接完成销售分层和目标调整判断
从"看数"直接过渡到"定动作"
用户
销售负责人 / 运营负责人 / 体验营项目负责人（不是一线销售）
不做
多页面经营分析平台 / 客户级明细下钻 / 预警规则配置后台
2. 顶部筛选区
期次 — 最核心筛选项，默认进入当前期次，全页图表/卡片/表格联动
销售 — 下拉单选，筛选特定销售视角
客户来源 — 下拉单选
时间范围 — 快捷项（近7天/近30天）+ 自定义日期区间
要求
筛选后全页所有模块同步联动，无需手动刷新
3. 经营概览（10 张卡）
第一行（规模 + 承接）
总添加人数 · 有效 leads · SA 客户数 · 领卡人数
第二行（执行 + 风险）
填问卷人数 · 直播间人数 · 入群人数 · 转化人数 · 退款人数 · 未回复人数
入群人数去重口径：同一客户加多个群只计 1 人
每张卡展示
指标名 + tooltip 说明 / 当前值 / 业务解读文案 / 环比 / 同比
要求
风险指标（退款、未回复）必须用风险色突出
卡片说明文案需体现管理意义，不重复字段名
4. 转化漏斗
漏斗层级（共 6 层）
总添加 → 有效 leads → 领卡 → 问卷 → 付款人数 → 成交
每层展示
当前人数 / 相对上一层转化率 / 流失量
要求
突出最大掉点环节；图表库不可用时显示 fallback 文案
漏斗层级须与经营概览、销售明细表关键字段保持一致
5. 转化趋势
X 轴：体验营期次；Y 轴：人数
展示序列：问卷数（核心结果）/ 转化数（核心结果）/ 退款数（风险）
用于判断节奏，不替代概览区的绝对值展示
6. 成交结构分析
A. 成交客户添加渠道
图表类型：双层嵌套环形图（旭日图）
内圈（一级渠道）：订阅号/公众号 · 视频号 · 网站 · 矩阵服务号 · 合作引流 · 深海圈 · 小程序 · 长尾
外圈（二级子渠道）：网站→索引站/搜索引擎；合作引流→用户分销/KOC导流；深海圈→AI编程/B站好物
右侧图例展示一级渠道名 + 成交数量；标签全部隐藏，hover 显示渠道名/总计/二级明细占比
圆心动态显示 hover 渠道名及成交量；底部注明「内圈为一级渠道，外圈为二级渠道」
B. 客户成交占比
图表类型：双轴柱状图（柱：成交单量；折线：转化率）
维度：按销售人员；判断个人成交贡献与转化效率
要求
两图并列同一层级；渠道图看来源结构，占比图看销售责任归属，不能混用
7. 销售作战视图
第一层：4 张诊断卡
销售团队总诊断 / 本期该冲谁 / 本期该救谁 / 本期该盯什么
所有结论必须明确到人或关键变量，不能只报数
第二层左侧：3 张目标卡
问卷目标 / 付款目标 / 执行目标
目标值由系统基于近 3 期均值自动测算，默认标注「系统估算」角标
每张卡必须展示：目标值 / 当前值 / 差额 / 追赶建议
点击目标数字弹出「按销售维度调整」弹窗，展示 4 位销售的个人目标 / 当前完成 / 差额 / 完成率
弹窗内个人目标支持行内编辑：点击数字直接输入，Enter/失焦保存，Esc 取消；合计行实时联动
执行目标弹窗合计行显示团队均值（而非求和），差额带 % 单位
弹窗关闭方式：点遮罩 / 右上角 ✕ / 按 Esc
第二层右侧：3 张分层卡（个人维度）
冲刺型销售 / 观察型销售 / 风险型销售
分层须与销售明细表经营分组列保持一致
第二层底部：团队执行风险摘要（团队维度）
独立展示，不与个人分层卡混排
必须分别列出：延期待办数 + 未完成预警数，不只展示合计
8. 销售数据明细
固定列
销售 · 经营分组 · 当前短板
关键结果（8列）
有效 leads · SA客户 · 领卡数 · 问卷数 · 直播间人数 · 进群人数 · 转化数 · 退款数
执行状态（3列）
预警处理 · 平均耗时 · 完成率
承接转化（3列）
领卡率 · 问卷率 · 付款率
收尾列
本期建议动作 · 目标调整建议
跳转规则
有效 leads / SA客户 / 领卡数 / 问卷数 / 转化数 支持点击跳转客户管理页，传入对应销售人员 + 客户状态过滤参数
直播间人数 / 进群人数 / 退款数 暂不支持跳转
要求
表格是责任归属表，不是纯数据表
经营分组用明显标签区分冲刺型（绿）/ 观察型（橙）/ 风险型（红）
建议动作和目标调整建议须支撑负责人直接安排动作
9. 非功能需求
筛选后全页联动，无需手动刷新
图表库不可用时降级显示 fallback 文案，不白屏
核心指标说明通过 hover tooltip 承载，不拆独立说明区
页面权限：仅负责人角色可访问，一线销售不可见
按销售调整目标
销售 个人目标 当前完成 差额 完成率
