标签管理 PRD v1.0
1. 产品背景
SCRM 中存在三种来源的客户标签，需要统一管理入口
企微标签：企微侧边栏手动打标，同步企微后台
CRM 标签：系统内部业务标签，不同步企微
自动化标签：CDP 标签引擎自动计算（规则+模型）
核心指标
标签定位时间 < 15 秒；标签创建 ≤ 3 步
2. 企微标签
左侧 — 标签组树（两级）
初始分组：转化组 / 交付组 / 运营组 / 来源组 / 默认分组（不可删除）
Hover 一级组：显示 新建子组 / 编辑 / 删除
Hover 二级组：显示 编辑 / 删除
点击「全部」→ 显示所有标签；点击一级组 → 该组全部；点击二级组 → 仅该子组
删除约束：分组下有标签时禁止删除；默认分组不可删除
右侧 — 标签列表
表格列：ID / 标签名称 / 人数（可点击）/ 操作（修改、删除）
新建标签 → 弹窗输入名称，归入当前选中子组（未选中则默认分组）
搜索：按标签名模糊过滤 | 分页：默认 5 条/页，支持 5/10/20/50
企微同步
SCRM ↔ 企微双向同步，CRUD 后 < 5 秒同步到企微
受企微 API 限制：最多 20 个标签组，每组最多 200 个标签
3. CRM 标签
结构和交互与企微标签完全一致
初始分组：销售阶段 / 客户价值 / 服务管理 / 运营分类 / 默认分组
与企微标签的区别
CRM 标签为系统内部管理标签，不同步企微
无标签组/标签数量限制
用途：销售/服务内部分类，不在企微侧边栏展示
4. 自动化标签
统计栏（4 个指标卡片）
总标签数 / 已启用规则 / 今日更新标签数 / 标签覆盖率
左侧 — 分类树（两级，只读）
一级：属性标签(44) / 行为标签(46)
属性子类：基础身份(14) / 交易属性(10) / 计算属性(12) / 偏好属性(8)
行为子类：内容消费(6) / 内容创作(4) / 学习行为(12) / 社交互动(7) / 项目参与(8) / 平台使用(9)
点击一级 → 该大类所有标签；点击子类 → 仅该子类
右侧 — 标签列表
表格列：标签名称（可点击→详情）/ 标签代码 / 取值范围 / 更新频率 / 关联规则 / 状态 / 操作
状态 Toggle：启用/禁用自动化标签
搜索：按名称或代码模糊过滤 | 分页：默认 10 条/页
标签详情 Drawer
点击标签名 → 右侧 Drawer 滑入（480px）
基本信息：名称 / 代码 / 分类 / 来源 / 频率 / 时间窗口 / 说明
取值逻辑：表格（取值 → 判定条件），如「高风险 → 续费倾向分 < 30」
关联规则：引用该标签的规则数 + 跳转链接
5. 校验规则
标签名称 / 分组名称：必填，不可为空
同一来源同一分组内标签名不可重名
同一级别内分组名不可重名
删除分组：下有标签时禁止；默认分组永远不可删除
6. 模块关系
人群包：标签条件的来源/分组/值下拉数据源来自此页面
自动化标签管理：本页是浏览入口，自动化标签管理是规则配置入口
客户管理：客户列表标签列 + 打标签操作调用此处数据
企微侧边栏：企微标签在侧边栏展示和操作
7. 交互规范
Tab 切换保留各 Tab 的筛选/选中/分页状态
分组树：选中高亮 + Hover 显示操作 + 搜索实时过滤
新建/编辑：居中弹窗输入名称
删除：confirm 二次确认
Drawer：右侧滑入 250ms，遮罩点击或关闭按钮关闭，滑出 200ms
Toast：所有操作结果居中提示，1.2 秒消失
