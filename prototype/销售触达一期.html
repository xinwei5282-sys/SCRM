1. 需求概述
背景
当前无统一触达入口，任务创建与执行分散
当前原型聚焦群发/私戳任务创建、列表查看与数据结果回看
通过统一入口沉淀发送通道、目标人群、推送内容与基础风控能力
一期目标
单页完成触达任务创建、修改与草稿保存
支持任务列表、数据页两类查看路径
在原型中明确呈现发送通道选择、人群圈选、内容录入、防骚扰过滤与结果回收
Out of Scope
人群规则配置器、模板变量渲染、A/B 测试、审批流
拦截记录页（已从一期删除，后续排期）
2. 创建任务 · 基础设置
必填字段
任务名称 — 手动填写，建议含活动名+日期
所属鱼丸 — 全部 / 按鱼丸选择
发送通道 — 企微官方数发 / 星云
交互规则
新建页采用单页表单，不再展示步骤条
发送通道以卡片形式展示，当前原型默认单选态呈现
所属鱼丸支持「全部鱼丸 / 部分鱼丸」切换；选择部分鱼丸时，点击输入区域打开企微组织架构弹层
组织架构弹层同时展示部门与企微成员；已选鱼丸以 tag 展示，支持逐个删除；删除后同步取消组织架构内对应勾选
发送通道、目标人群、推送内容、拉黑/删除过滤内容区收窄展示，保持左对齐
任务详情（只读模式）
列表「详情」入口复用创建页，切换为只读态：所有输入控件禁用，footer「保存草稿」与「保存」按钮同步禁用
只读态下顶部返回按钮文案改为「返回列表」，编辑态下显示为「取消」
只读态与编辑态共用同一页面，通过模式标记切换，不新增独立路由
3. 创建任务 · 目标人群
展示内容
条件编辑区：字段 / 运算符 / 标签值 / 且或非连接关系
底部仅保留「添加条件」操作
可选字段共 13 个，见下方穷举
企微标签：运算符「满足任一 / 同时满足 / 全部不满足」；值区为标签自由输入（支持多个，chip 展示）
客户等级：运算符「等于 / 不等于」；值区下拉单选：S / A / B / C
是否领卡：运算符固定「等于」（不可更改）；值区下拉单选：是 / 否
是否是官网会员：运算符固定「等于」（不可更改）；值区下拉单选：是 / 否
是否加入企微群聊：运算符固定「等于」（不可更改）；值区下拉单选：是 / 否
是否观看直播：运算符固定「等于」（不可更改）；值区下拉单选：是 / 否
是否分销：运算符固定「等于」（不可更改）；值区下拉单选：是 / 否
体验营期数：运算符「等于 / 不等于」；值区下拉单选：1期 / 2期 / 3期 / 4期 / 5期
首次加入时间：运算符「等于 / 开始于 / 结束于 / 为空 / 不为空」；值区为日期输入（YYYY-MM-DD）；选「为空 / 不为空」时值区隐藏
添加好友时间：运算符「等于 / 开始于 / 结束于 / 为空 / 不为空」；值区为日期输入（YYYY-MM-DD）；选「为空 / 不为空」时值区隐藏
所属销售：运算符「等于 / 不等于 / 包含」；值区为销售人员下拉（系统成员列表）
分销人：运算符「等于 / 不等于 / 包含」；值区为销售人员下拉（系统成员列表）
浏览文章时长：运算符「大于 / 小于 / 大于等于 / 小于等于 / 等于 / 不等于」；值区为数值输入（单位：分钟）
操作
预计命中人数移至 footer 展示
footer 展示命中人数与「刷新估算」按钮
4. 创建任务 · 推送与风控
推送内容
手动填写文本，建议控制在 1000 字以内
内容快照会完整存档到任务详情页
支持上传图片、视频，以及通过弹窗录入链接、小程序、视频号信息
顶部快捷变量入口展示「昵称 / 备注」两个可插入占位符
发送方式
立即发送 — 提交后立即执行
定时发送 — 必填发送时间，限 09:00–21:00
防骚扰设置（固定，不可自定义）
原型页当前显式展示「拉黑/删除客户过滤」且始终开启、不可关闭
footer
按钮居中展示：取消 / 保存草稿 / 保存
footer 固定在新建页底部，且与内容区底边对齐
顶部标题不固定，跟随内容区一起滚动
新建页最大展示高度为一屏，超出内容仅在中间内容区滚动
推送内容为空时不允许提交
5. 高优先 & 已超时判断
当前原型状态
页面未显式展示高优先、已超时标签和排序规则
本节保留为后续补充项，尚未在群发/私戳原型主界面落地
6. 任务列表
页面入口
页头右上提供「＋ 新建任务」入口，进入新建触达任务页
列表字段
任务名称 / 创建人 / 发送方式 / 计划时间 / 状态 / 预计触达 / 实际触达 / 回复人数 / 回复率
筛选项
筛选栏：搜索任务名称 + 重置（状态筛选已改为下方 Tab）
Tab 筛选（搜索栏下方）：全部 / 草稿 / 待执行 / 执行中 / 已完成 / 部分失败
操作项（按状态区分）
草稿：修改 / 删除
待执行：修改 / 复制 / 删除
执行中 / 已完成 / 部分失败：数据 / 详情 / 复制
「数据」进入结果数据页；「详情」进入任务详情只读页（复用创建页只读态）
底部提供分页区：上一页 / 页码 / 下一页 / 每页条数
7. 数据详情
页面结构
页头展示统一返回 icon、页面标题「数据」以及顶部「复制任务」按钮
页面主体由摘要区、KPI 区（常驻）+ 三个 Tab（客户明细 / 成员明细 / 任务日志）组成
Tab 结构
客户明细（默认展示）：客户列表 + 筛选工具栏 + 导出
成员明细：成员执行明细表 + 执行观察（两个子模块同 Tab 展示）
任务日志：按时间线展示关键节点
Tab 切换时摘要区与 KPI 区保持常驻，仅下方面板内容切换
顶部指标
标题与状态同一行：任务名称 / 已完成
摘要字段一行 4 个：创建人 / 发送方式 / 创建时间 / 计划时间 / 选择鱼丸 / 推送内容
KPI：命中人数 / 排除人数 / 成功人数 / 回复人数
客户明细
字段：客户昵称 / 星球编号 / 所属销售 / 触达状态 / 是否回复 / 回复时间 / 排除原因
客户昵称列展示头像、客户名称，第二行展示手机号；无手机号显示 -
所属销售列第二行展示鱼丸号
客户明细工具栏支持触达状态筛选、回复状态筛选、昵称/星球编号搜索，以及导出客户明细
其他信息
成员执行明细：销售姓名 / 分配人数 / 已执行 / 回复人数
执行观察：完成率、高回复成员、失败集中原因
任务日志：按时间线展示创建、命中计算、执行开始、执行完成、结果回流
