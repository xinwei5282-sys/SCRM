## 3. 交互设计

### 3.1 页面流程

```
首页加载
 ├─ 自动弹出新手任务弹窗（延迟 600ms）
 │
 弹窗内交互
 ├─ 点击"去添加/去填写" → 弹窗内完成任务，进度条实时更新
 ├─ 全部完成 → 弹窗切换为庆祝态 + 彩纸动画
 │   └─ 点击"开始探索" → 关闭弹窗，回到首页正常浏览
 └─ 点击关闭(×) / Escape → 关闭弹窗，显示悬浮提醒条

 悬浮提醒条
 ├─ 点击文字区域 → 重新打开新手任务弹窗
 └─ 点击关闭(×) → 隐藏提醒条，状态同步到服务端

 全部完成后
 └─ 提醒条不再显示，弹窗不再弹出
```

### 3.2 交互动效

| 场景 | 动效 | 时长 | 缓动 |
|------|------|------|------|
| 弹窗弹出 | 淡入 + scale 0.92→1 + translateY 20→0 | 350ms | cubic-bezier(0.34,1.56,0.64,1)（定制弹性动画） |
| 弹窗关闭 | 淡出 | 300ms | ease |
| 提醒条展开 | max-height + opacity | 400ms / 300ms | ease |
| 提醒条关闭 | max-height + opacity | 400ms / 300ms | ease |
| 任务完成动画 | scale 1→1.02→1 + border 变绿 | 450ms | ease |
| 进度条增长 | width 过渡 | 500ms | cubic-bezier(0.25,0.8,0.25,1) |
| Hover 颜色变化 | 背景色/边框色/文字色 | 100ms | ease（Nova 标准） |
| 头部 emoji 浮动 | translateY ±6px | 2500ms | ease-in-out infinite |
| 庆祝 emoji 晃动 | scale ±0.1 + rotate ±5deg | 1000ms | ease-in-out infinite |
| 彩纸飘落 | translateY + rotate 720deg | 1.5-3s random | linear |
| Toast 弹出 | translateY -20→0 + opacity 0→1 | 300ms | ease |
| Toast 消失 | opacity 1→0 | 300ms | ease |

## 4. 状态设计

### 4.1 加载态

| 场景 | 设计 |
|------|------|
| 弹窗内任务列表首次加载（> 500ms） | 2 个任务卡片骨架屏占位（灰色矩形 shimmer 动画） |
| 任务操作按钮 loading | 按钮内显示 loading spinner（16px），文字隐藏，按钮不可重复点击 |
| 二维码弹窗加载中 | 弹窗内 200×200px 灰色占位 + 居中 spinner |

### 4.2 错误态

| 场景 | 设计 |
|------|------|
| 任务列表加载失败 | 列表区显示"加载失败" + "点击重试"按钮，居中展示 |
| 任务完成操作失败 | Toast 提示"操作失败，请检查网络后重试"，按钮恢复为可点击状态 |
| 二维码加载失败 | 弹窗内占位区显示"加载失败，点击重试"文字 |

## 7. 关键设计决策

| 决策 | 说明 |
|------|------|
| 弹窗内完成 | PC 端新手任务在弹窗内直接完成，不跳转独立页面，减少页面跳转提高完成率 |
| 仅 2 个任务 | PC 端去掉移动端专属的"绑定星球账号"，保留跨端通用任务 |
| 弹窗 520px | 比移动端 300px 更宽，容纳任务卡片+图标+按钮的完整横向布局 |
| 品牌主色按钮 | 任务操作按钮使用 `--color-primary-6`，与移动端保持一致 |
| CTA 用橙色 | 弹窗头部渐变保持 `--gradient-accent-full` 橙色，突出行动号召 |
| 庆祝切绿色 | 全部完成后头部切换为 teal 渐变，传递完成感和品牌归属感 |
| 提醒条唤回 | 关闭弹窗后通过提醒条持续提醒，点击可重开弹窗，保证任务触达率 |
| 整合新人登记表 | 复用现有功能入口和数据，避免重复开发和用户重复填写 |
| Web 端字体 | 使用 §3.4 Web 端文字规范，字号整体大于移动端（如标题 22px vs 移动端 18px） |
| Nova 圆角统一 | 按钮 `--radius-btn` (6px)，卡片/弹窗 `--radius-card` (8px) |
| Nova 遮罩统一 | 弹窗遮罩 `--color-overlay` (80% 黑色) |
| 定制弹窗动画 | PC 弹窗使用弹性动画（350ms cubic-bezier），有意偏离 Nova 标准 200ms，提升品牌感 |
| 键盘交互 | 支持 Escape 关闭、Tab 焦点陷阱、首次打开焦点落在操作按钮 |
