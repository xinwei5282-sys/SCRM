# 企微侧边栏 UI 设计方案

> 设计基础：`.claude/design-foundation.md`（颜色、字体、间距、圆角、组件规范等所有 Token 定义均以该文件为准）
> 设计语言：Nova 紧凑布局（容器 padding 16px, gap 16px, btn 32px, radius-btn 6px, radius-card 8px）
> 版本：v1.0 | 日期：2026-03-06
> PRD 对应：[PRD_企微侧边栏.md](PRD_企微侧边栏.md) v4.0

## 1. 整体框架

### 1.1 侧边栏容器

```
┌─────────────────────────────────────┐
│  Header  标题 + 操作按钮      40px   │ --color-fill-3
├─────────────────────────────────────┤
│  Nav Tabs  3 Tab 导航        ~39px  │ --color-fill-3
├─────────────────────────────────────┤
│  [预警横幅 — 仅预警态]               │ --color-danger-1
├─────────────────────────────────────┤
│                                     │
│  Body  滚动内容区  (flex: 1)         │ --color-fill-1
│                                     │
├─────────────────────────────────────┤
│  Footer  底部操作栏            52px  │ --color-fill-1
└─────────────────────────────────────┘
```

| 属性 | 值 |
|------|-----|
| 宽度 | 360px |
| 高度 | 700px（原型高度，实际由企微 WebView 决定） |
| 背景 | `--color-fill-1` |
| 圆角 | `--radius-card` (8px) |
| 阴影 | `0 8px 40px rgba(0,0,0,0.12), 0 0 0 1px rgba(0,0,0,0.04)` |
| 布局 | `display: flex; flex-direction: column` |

### 1.2 Header

| 属性 | 值 |
|------|-----|
| 高度 | 40px |
| 背景 | `--color-fill-3` |
| Padding | 0 12px |
| 底部边框 | 1px solid `--color-border-default` |
| 标题 | 13px / 600 / `--color-text-2` |
| 操作按钮 | 24×24px, `--radius-xs`, hover: `--color-fill-3-hover` |

销售组标题：**CRM 客户管理** — 操作：搜索、更多
交付组标题：**CRM 会员运营** — 操作：导出、设置

| 属性 | 值 |
|------|-----|
| 容器 Padding | 6px 16px |
