## 命名规范

- 索引文件：`PRD.md`（本文件）
- 模块 PRD：`PRD_功能名.md`
- UI 设计：`UI_Design_功能名.md`
- HTML 原型：`prototype/功能名.html`

## 新增 PRD 流程

1. 创建 `PRD_功能名.md`，按标准模板编写
2. 在本文件的 PRD 清单中添加一行
3. 如有对应 UI 设计，创建 `UI_Design_功能名.md`
4. 如有原型，创建 `prototype/功能名.html`
